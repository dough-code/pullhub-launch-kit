Privacy Policy

How Pullhub handles local boards, shared boards, Google access, and subscription-related data.

Effective date: June 1, 2026

Pullhub is provided by debutt studio. Contact: support@debutt.studio.

What Pullhub does

Pullhub helps users save visual references into local boards, capture screenshots, export backups, share boards when enabled, and push selected images into Google Slides.

Information we process

Pullhub may process image URLs, screenshots or image data selected by the user, page URLs associated with saved references, board names, board notes, Google account email and profile photo after sign-in, Google Slides and Drive identifiers needed to push images, subscription status, and optional shared-board data.

How information is used

Information is used only to provide Pullhub features: saving boards, showing saved references, creating backups, sharing boards, authenticating Google access, pushing images to Google Slides, and managing subscription access.

Where information is stored

Most board data is stored locally in the browser using Chrome extension storage. If board sharing is used, shared-board data may be stored in Firebase/Firestore. Images pushed to Google Slides may be uploaded to Google Drive when required so Google Slides can access them.

Google Drive image fallback

When direct insertion into Google Slides fails, Pullhub may upload the image to the user's Google Drive folder named "Pullhub Uploads". The uploaded file may be set so anyone with the link can view or read it. This permission is required so Google Slides can access and insert the image. These files remain in the user's Google Drive and can be managed or deleted by the user.

Third-party services

Pullhub uses Google APIs for authentication, Drive, and Slides features; Firebase/Firestore for optional board sharing; and ExtensionPay/Stripe for subscription management.

The Chrome Web Store listing permission description should match the Google Drive image fallback disclosure in this policy.

Google API Limited Use

Pullhub's use and transfer of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

What we do not do

We do not sell user data. We do not use user data for personalized advertising. We do not access browsing activity except as needed for user-facing Pullhub features such as saving the current page URL, capturing selected page content, or detecting Google Slides tabs.

Data deletion

Users may delete boards inside the extension, clear local browser extension data, revoke Google access from their Google Account, or contact support@debutt.studio for help with shared-board or account-related deletion requests.

Changes

We may update this policy as Pullhub changes. Material changes will be reflected on this page.
